# Tetris

A classic Tetris game playable directly in the browser.

🎮 **Play:** https://tetris.mbz-ilvesheim.de

## Controls

| Key | Action |
|-----|--------|
| ← → | Move |
| ↓ | Soft drop |
| ↑ / X | Rotate clockwise |
| Z / Ctrl | Rotate counter-clockwise |
| Space | Hard drop |
| P | Pause |

## Development

Single-file HTML/CSS/JS — no build step required. Just open `index.html` in any modern browser.
